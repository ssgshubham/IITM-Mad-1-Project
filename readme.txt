Step 1: Extract the Code folder, and open it in terminal

Step 2: Create a virtual environment, run the following commands in terminal: 

Command 1: pip install virtualenv

Command 2: python -m virtualenv demoEnv

Command 3: demoEnv\Scripts\activate

Step4: Run the app.py file, run the following command in terminal:

Command 4: python app.py

Step5: In the terminal you'll get a link, ctrl + click on it and go to the website
